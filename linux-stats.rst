=======================
Linuxパフォーマンス関連
=======================

.. highlight: console

vmstatでざっくり調べる
======================

2秒ごとに ``vmstat`` を取得する(単位はMB)::

	$ vmstat -w -S m 2

procs
	r
		実行キューに入っているプロセス数

	b
		実行可能だがブロックされているプロセス数

procsのr数値が高い
------------------

実行中のプロセスは ``ps`` コマンドの *STAT* フィールドで **R** が付いている。
**R** フラグは、実行中またはrun queueに入っているものに付くので、
run queueのみを調べる方法はなさそう。

	$ ps -ef

``top`` でも調べられるらしいがよくわからなかった。
